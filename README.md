## How to Run

1. Clone this repository to your local machine:

   ```bash

   ```

2. Navigate to the project directory:

```bash

cd caesar-cipher-python

```

3. Run the project:

```bash

python caesar_cipher.py

```

4. Follow the on-screen instructions to enter the message and shift value.
